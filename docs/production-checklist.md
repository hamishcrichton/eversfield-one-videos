# Production Checklist

Per-video checklist. Copy this for each new video and track progress.

## Pre-Production

- [ ] Script written and reviewed (`videos/XX/script.md`)
- [ ] Storyboard created (`videos/XX/storyboard.md`)
- [ ] Total word count confirms target duration (~100 words/minute)

## Character & Voice

- [ ] Character pack generated (6+ images in `assets/character-pack/`)
- [ ] Character images reviewed for consistency across poses
- [ ] Character images upscaled to 2048x2048
- [ ] ElevenLabs voice selected and tested
- [ ] Voice audio generated per section (`videos/XX/voice/`)
- [ ] Voice audio reviewed for pacing and pronunciation

## Screen Recording

- [ ] Demo account prepared with realistic data
- [ ] Browser configured (Chrome, no bookmarks bar, 1920x1080)
- [ ] System notifications hidden
- [ ] All screen recordings captured per storyboard
- [ ] Recordings reviewed for smooth mouse movement and timing

## Animation

- [ ] RunPod pod launched (RTX 4090)
- [ ] ComfyUI running with all models loaded
- [ ] Lip-sync clips generated for all narrated sections
- [ ] Lip-sync clips reviewed for sync quality
- [ ] Dynamic scene clips generated (opening, transition, closing)
- [ ] Scene clips reviewed for character consistency
- [ ] Scene clips upscaled to 1080p
- [ ] RunPod pod stopped (to save costs)

## Post-Production

- [ ] DaVinci Resolve project created
- [ ] All assets imported (screen recordings, lip-sync clips, scene clips, voice audio)
- [ ] Timeline assembled per storyboard structure
- [ ] Character PiP positioned and sized correctly
- [ ] Text overlays added (section titles, CTA info)
- [ ] Background music added and levelled (-18 to -24dB)
- [ ] Transitions smooth between sections
- [ ] Full playthrough review — no jarring cuts or sync issues
- [ ] Colour grading consistent across all segments
- [ ] Export: H.264, 1920x1080, 30fps

## Publishing

- [ ] YouTube thumbnail exported (1280x720)
- [ ] YouTube title written (under 60 characters)
- [ ] YouTube description written with timestamps
- [ ] Tags added (fire door compliance, building safety, etc.)
- [ ] Video uploaded and published
